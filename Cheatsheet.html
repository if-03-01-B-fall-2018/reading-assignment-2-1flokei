HTML

1. Name the components of html elements properly
Starting Tag,Content and Ending Tag; examples for starting tags:h1,p,br,
2. Identify void elements and how they are denoted
Void elements don't have content inside of them and the also don't have an ending-tag. An example is the <br> tag (but can technically be closed this way: "<br />")
3. Identify attributes
href,src,width,height,alt,style,lang,title,id,disabled
4. Write down the correct DOCTYPE declaration for html 5
<!DOCTYPE html>
5. Write down syntactically correct statements for the following elements
  1. Article: <article><h1>First blogpost</h1></Article>
  2. Body: <body><p>This is text</p></body>
  3. line breaks: <br>
  4. Headings:
                      <h1>Heading 1</h1>
                  <h2>Heading 2</h2>
                  <h3>Heading 3</h3>
                  <h4>Heading 4</h4>
                  <h5>Heading 5</h5>
                  <h6>Heading 6</h6>
  5. Section with meta-information: 
                    <section>
                  <meta charset="UTF-8">
                </section>
  6. Top level element
  7. Image named
  8. Paragraph
  9. Section: <section></section>
  10. Label appearing in the title bar: <title>Pagetitle</title>
  11. Link to other pages: <a href="http://duckduckgo.com"> "Zur Suchmaschine"</a>
  12. Comment: <!-- This is a comment -->;
  13. Head with title and meta elements 
                    <head>
                  <title>Pagetitle</title>
                  <meta name="Author" content="Florian Keintzel">
                </head>

CSS

1. Name the components of a css rule properly:
Selector and declaration block with properties in it (decklaration in {},and every property ends with ';')
2. Use combinators properly: direct child, descendant, adjacent sibling, sibling 
          descendant:
          div h1{
            text-align: center;
          }

          child:
          div > h1 {
            text-align: right;
          }

          adjacent sibling Selector
          div + h1 {
            text-align: left;
          }

          general sibling Selector
          div ~ h1 {
            background-color: green;
          }
3. Specify colors by color names and in hexadecimal notation
  1. Red: #FF0000
  2. Green: #00FF00
  3. Blue: #0000FF
  4. White: #FFFFFF
  5. Black: #000000
4. Declarations for color, background color
style="color:MediumSeaGreen;" || style="background-color:blue;"
5. Pseudo classes for the <a> element
a:link,a:visited,a:hover,a:active
6. Declaration for background image
background-image: [color] (valid color name e.g. "red", hex value e.g. ff0000, rgb value e.g. rgb(255,0,0)
7. Declaration for text alignment (left, right, center, justify)
text-align: [left,right etc]
8. Declaration for text decoration (overline, underline, line-through)
text-decoration: [overline,underline etc (can also be: none)]
9. Declaration for text transformation (uppercase, lowercase, capitalize)
text-transform: [uppercase, lowercase etc]
10. Identify the difference between serif and sans-serif fonts
Sans means without, therefore sans serif doesn't have those small lines at the end of some characters that serif has
11. Declarations for font families
font-family: "this is font 1",font2,font3; (if the first one is not available the second comes and so on..)
12. Declarations for font style normal and italic
font-style: normal; || font-style: italic
13. Declarations for font sizes
font-size: [number]px (e.g. font-size: 40px)
14. Declarations for font weights
font-weight: [normal,lighter,bold]
